Setting Up Help Pages¶

I’m on a Mac: I have my user folder, in which I have an AllHelp folder containing subfolders for each product. When I create a new product and wish to create help for that product, I create a folder in my AllHelp folder.

I then create a repository for that product (call it “MyApp”) on GitHub. After that:

  • cd AllHelp/MyApp
  • run sphinx-quickstart

In the list of prompts, it will ask whether to separate source from build: select Yes. In order to have the build end up in the right place, I have the following entry in the Makefile:

* BUILDDIR      = /Users/robinhahn/AllHelp/GitPages/CallSheet

In the index.rst, change the title as needed, then add the pages in the RSTs folder so:

.. toctree::
   :maxdepth: 3
   :caption: Contents:


   RSTs/Introduction.rst
   RSTs/GetStarted.rst
   RSTs/Licence.rst
   RSTs/Caveats.rst

Modify the conf.py for sphinx to use a nicer template:

html_theme = 'sphinx_rtd_theme'

html_theme_options = {
    'canonical_url': '',
    'analytics_id': '',
    'logo_only': False,
    'display_version': True,
    'prev_next_buttons_location': 'bottom',
    'style_external_links': False,
    #'vcs_pageview_mode': '',
    # Toc options
    'collapse_navigation': True,
    'sticky_navigation': True,
    'navigation_depth': 4,
    'includehidden': True,
    'titles_only': False
}

You will need two instances of Terminal running: one for building html from .rst files, the other, to talk to github.

After adding and editing your files to the RSTs folder, edit your index.rst to include all these files. Do a preliminary make html to check for errors.

In the Second Terminal (the one for git), invoke git:

  • cd /to/working/git/folder
  • git init
  • git clone git@github.com:robinboncoeur/xlVBA-AssetMgr.git html
  • cd html
  • git symbolic-ref HEAD refs/heads/gh-pages
  • rm .git/index
  • git clean -fdx

At this point, the folder is clean. So, run:

* make html

Then:

  • git add -A
  • git commit -am “-date- committing help”
  • git push origin gh-pages

From then on, you need to navigate to the html folder

  • cd /AllHelp/GitPages/MyApp/html
  • git add -A
  • git commit -am “-date- committing help”
  • git push origin gh-pages

Saves:

cd /Users/robinhahn/AllHelp/GitPages/AssetMgr/html
cd /Users/robinhahn/AllHelp/GitPages/CallSheet/html
cd /Users/robinhahn/AllHelp/GitPages/FigureShader/html
